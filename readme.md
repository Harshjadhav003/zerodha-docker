# 🚀 ZERODHA

A full-stack stock trading platform inspired by Zerodha, built using a microservices architecture. The project demonstrates modern software engineering practices including secure authentication, real-time communication, Dockerized deployments, caching, and scalable backend systems.

## 🌟 Highlights

* 🔐 JWT Authentication & Authorization
* 📈 Trading & Portfolio Management
* 📊 Interactive Analytics Dashboard
* ⚡ Redis Caching
* 🔄 Real-Time Updates with Socket.IO
* 🐳 Dockerized Infrastructure
* ☁️ Cloud Deployment on Vercel & Render
* 🏗️ Microservices-Based Architecture

## 🛠️ Tech Stack

**Frontend:** React.js, Vite, Axios

**Backend:** Node.js, Express.js, JWT, Socket.IO

**Database & Cache:** MongoDB, Redis

**DevOps:** Docker, Docker Compose, Nginx

**Deployment:** Vercel, Render
